Instructions
===============================================================================

FIRST TIME: need to be using `venv` , and then install stuff and save requirements:

	virtualenv venv
	source venv/bin/activate
	
	pip install <stuff>
	pip freeze > requirements.txt



THEN! don't forget to activate virualenv:

	source venv/bin/activate

        deactivate
