Ranking Countries

We're all familiar with various rankings of the world's universities. Recently, however, US News came out with a ranking of the world's countries. Partly in order to investigate their methodology, this project tried to recreate the US News ranking, to contrast what is shown by propriatary survey data (the basis for the US News ranking) with hard, real metrics that are publicly accessible (for this ranking).

However, not being able to replicate some of their input features, in this app, you can instead create your own country ranking and explore the data.

Data Details

My data was sourced from 3 locations:

  • The World Bank - Indicators
  • World Economic Forum - Global Competitiveness Index
  • WIN/Gallup International - End of the Year Survey

    • ...which created a problem: different organizations use different country names. I resorted to regex to standardize them.

    Tech Stack

    Briefly, this is a Python-flask web application with a PostgreSQL database, all deployed to Heroku. Data for the Data Explorer was pre-processed with Pandas, and the output is a Bokeh plot. Other data munging was done with R, and the plot from Week 1 was done in D3, based on a template from NVD3. Page styling was done with Bootstrap (plus some minor CSS and HTML tweaks - Safari console debugger was very useful!). Everything was packaged together with Flask (forms use Flask-WTF), and deployed on Heroku. Favicon is my own, created on Paper for iOS and converted with favicon.cc
